# Keycloak
## About

Project page: [https://www.keycloak.org/](https://www.keycloak.org/)

Repository URL: [https://github.com/keycloak/keycloak](https://github.com/keycloak/keycloak)

## Components

Components

### Keycloak Server

Keycloak Server is the main component of Keycloak. It provides authentication and authorization services for applications.

### Keycloak Admin Console

Keycloak Admin Console is a web-based interface for managing Keycloak instances. It allows administrators to create and manage users, roles, clients, and realms.

### Keycloak CLI

Keycloak CLI is a command-line interface for managing Keycloak instances. It allows administrators to create and manage users, roles, clients, and realms.

### Keycloak Docker Image

Keycloak Docker Image is a Docker image for running Keycloak in a container. It allows administrators to easily deploy and manage Keycloak instances.

### Keycloak Helm Chart

Keycloak Helm Chart is a Helm chart for deploying Keycloak on Kubernetes. It allows administrators to easily deploy and manage Keycloak instances.

### Keycloak Operator

Keycloak Operator is a Kubernetes operator for managing Keycloak instances. It allows administrators to easily deploy and manage Keycloak instances.
